## Features

### Extensible

Sittly is extensible with plugins. You can create your own plugins or use plugins created by the community.

### Open Source

Sittly is open source and free to use. You can contribute to the project by creating plugins or reporting bugs.

### Fast

Sittly is fast. It uses [Tauri](https://tauri.app/), [Typescript](https://www.typescriptlang.org/) and [React](https://es.react.dev/) to create a native app with web technologies.

### Navegable

We know how much money you spent on your keyboard, so we made Sittly navegable with it. _Don't worry you still can use the mouse_
