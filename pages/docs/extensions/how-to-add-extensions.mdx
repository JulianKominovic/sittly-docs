## How can I add extensions?

~You can search and install extensions from the **extension store**.~

<Callout type="warning">
  The extension store is not implemented yet. But...
</Callout>

You can copy github's repo url and paste it in the search bar, you will see this option ⤵️.

![Sittly add extension](assets/images/sittly-add-extension.png)

After clicking or pressing enter the extension will be installed.

Give it a try! Copy this url and paste it in the search bar.

https://github.com/JulianKominovic/sittly-emoji-extension/
