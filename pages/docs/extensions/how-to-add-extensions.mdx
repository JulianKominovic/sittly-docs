## How can I add extensions?

You can search and install extensions from the **store**.

Select `Store` from the list, hit `Enter` or `Click`.
![](assets/images/sittly-store.png)
Search for the extension you want to install.
Hit `Enter` or `Click` on the extension you want to install.
